# @ppos/printprice-os-bootstrap

This repository contains the initialization and bootstrapping logic for the **PrintPrice OS Platform**.

## Contents

- `bootstrap-repos.ps1`: Automated script to initialize the polyrepo environment.
- `mysql-init/`: Canonical database schema and migration baseline.
- `observability/`: System-wide telemetry collector and monitoring configurations.

## Getting Started

To initialize the local development environment:
```powershell
.\bootstrap-repos.ps1
```
